# Динамическая фото-галерея

Веб-приложение на PHP, которое позволяет загружать, просматривать и управлять изображениями без использования базы данных. Проект выполнен в рамках лабораторной работы по веб-разработке.

## Технологии
- PHP 7+
- HTML5 / CSS3
- JavaScript (Vanilla JS)
- JSON для хранения данных
- XAMPP / Apache в качестве локального сервера

## Как запустить проект
1. Установите XAMPP (или другой сервер с поддержкой PHP).
2. Склонируйте репозиторий в папку `htdocs`:
   ```bash
   git clone https://github.com/UnicornSereJa/photo-gallery.git
3. Запустите Apache в XAMPP Control Panel.
4. Перейдите в браузере по адресу:
    http://localhost/photo-gallery/

## Возможности проекта
- Проверка формата файла и защита от дубликатов
- Автоматическое создание уменьшенной копии (превьюшка)
- Добавление водяного знака
- Пагинация
- Удаление изображений
- Поиск по описанию
- Drag & Drop загрузка
- Адаптивная сетка галереи
- Hover-анимации карточек
